# Knowledge Graphs - Проект по графам знаний

## Описание файлов проекта

### 1. `ontology.rdf`
Файл онтологии с базовой структурой (классы, свойства, отношения) для представления знаний об автомобилях.

### 2. `cars_ontology.owl`
Файл графа знаний со всеми сущностями (автомобили, производители, типы кузова и т.д.) и их связями, созданный на основе онтологии.

### 3. `populate_ontology.py`
Код на Python с использованием библиотеки owlready2 для заполнения онтологии данными из CSV-файла и создания графа знаний.

### 4. `create_kg_embeddings_pykeen.py`
Скрипт обучения Knowledge Graph Embeddings (KGE) с использованием библиотеки PyKEEN для создания векторных представлений сущностей и отношений графа знаний.
