# Граф знаний по Honkai Star Rail

## Структура проекта

- `data/`: RDF-файлы с онтологией (hsr_ontology.rdf - базовая онтология, hsr_ontology_full.rdf - файл со всеми триплетами)
- `parsers/`: Скрипты для парсинга данных из игры
- `main.py`: Основной скрипт для работы с онтологией
- `train_hsr_embeddings.py`: Скрипт для обучения эмбеддингов
- `Honkai Star Rail.pptx`: Презентация проекта
