# Data Layout

## Назначение

`data/` хранит сырье, разметку и метаданные по видам птиц.

## Папки

- `raw/text/` - исходные текстовые описания (по видам).
- `raw/images/` - исходные изображения.
- `raw/audio/` - исходные аудиозаписи.
- `annotations/text/` - результаты текстовой разметки.
- `annotations/images/` - результаты разметки изображений.
- `annotations/audio/` - результаты аудиоразметки.
- `metadata/species_catalog.csv` - реестр видов и статусов готовности.
