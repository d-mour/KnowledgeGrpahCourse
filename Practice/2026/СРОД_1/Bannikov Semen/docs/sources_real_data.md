# Реальные источники данных по птицам

Ниже список источников, которые подходят для ЛР1-ЛР6 и дают реальные данные.

## Текстовые данные

1. Wikipedia (RU/EN) - энциклопедические описания видов
  - [https://ru.wikipedia.org/](https://ru.wikipedia.org/)
  - [https://en.wikipedia.org/](https://en.wikipedia.org/)
2. eBird Species (Cornell Lab) - описания и справка по видам
  - [https://ebird.org/species](https://ebird.org/species)

## Изображения

1. Wikimedia Commons - фото птиц (проверять лицензию каждого файла)
  - [https://commons.wikimedia.org/](https://commons.wikimedia.org/)
2. iNaturalist - наблюдения с фотографиями
  - [https://www.inaturalist.org/](https://www.inaturalist.org/)

## Аудио

1. Xeno-canto - записи голосов птиц
  - [https://xeno-canto.org/](https://xeno-canto.org/)
2. Macaulay Library (eBird/Cornell)
  - [https://www.macaulaylibrary.org/](https://www.macaulaylibrary.org/)

## Таксономия и идентификаторы

1. GBIF Species API - таксономические ID и справочная информация
  - [https://www.gbif.org/](https://www.gbif.org/)
  - [https://api.gbif.org/](https://api.gbif.org/)
2. Wikidata - дополнительные идентификаторы и связи
  - [https://www.wikidata.org/](https://www.wikidata.org/)
