# Битва при Ватерлоо — текстовый граф знаний

Берём статью из Википедии про Ватерлоо и превращаем текст в RDF-граф.

## Файлы

- **dataset.json** — 5 предложений из статьи, разбитых на атомарные утверждения с объектами
- **ontology.ttl** — схема: классы (Sentence, AtomicStatement, Object) и связи между ними
- **build_graph_colab.ipynb** — ноутбук, который собирает всё это в RDF
- **output/knowledge_graph.ttl** — результат (236 триплетов)

## Как это работает

Текст -> предложения -> атомарные утверждения (факты) -> объекты (сущности)

## Запуск

Открываем `build_graph_colab.ipynb` в Google Colab, запускаем — получаем `knowledge_graph.ttl`.

## Пример

Из предложения "Битва при Ватерлоо произошла 18 июня 1815 года" получается:

```
ex:sentence_1_atomic_0 a ex:AtomicStatement ;
    rdfs:label "Битва при Ватерлоо произошла 18 июня 1815 года" ;
    ex:mentionsObject ex:object_18_июня_1815_года,
        ex:object_Битва_при_Ватерлоо .
```
